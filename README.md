# Dong-Ha Shin Portfolio

A personal portfolio website for Dong-Ha Shin

**Live Demo:** [https://edengloria.github.io/dong-ha_page/](https://edengloria.github.io/dong-ha_page/)

## Tech Stack

- Next.js with TypeScript
- Tailwind CSS
- Framer Motion
- GitHub Pages

## Features

- Responsive design for all devices
- Holographic card UI effects
- Canvas-based beam animation background
- Performance optimization for mobile
- Dark mode

## Sections

- About
- Research Interests
- Publications
- Experience
- Patents
- Awards
- Gallery
- Contact

## Setup

```bash
# Clone repository
git clone https://github.com/edengloria/dong-ha_page.git

# Install dependencies
cd dong-ha_page
npm install

# Run development server
npm run dev
```

## Deployment

```bash
# Build static files
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Custom Domain Setup

To use a custom domain with GitHub Pages:

1. Configure DNS at your domain provider
2. Set custom domain in repository settings
3. Create a `CNAME` file in the repository root
